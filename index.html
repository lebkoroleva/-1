Что такое информация

Сведения об окружающем мире и процессах, которые:

  • воспринимаются
  • передаются
  • хранятся
  • обрабатываются

и помогают принимать решения

📌 Базовое понятие науки

Свойства информации

  • Объективная — независимая от мнения
  • Достоверная — отражает истину
  • Полная — достаточная для понимания
  • Актуальная — своевременная
  • Полезная — помогает в решении
  • Понятная — доступна для восприятия
📝 Ключевая тема ЕГЭ

Информация ≠ данные

Данные — числа, символы, сигналы

Информация — осмысленные и полезные данные

«37» → данные
«Температура 37°C — опасно» → информация

В разных областях

🏛️ Философия отражение реальности
🏠 Быт полезные сведения
📡 Техника сигналы и сообщения
🤖 Кибернетика средство управления
💻 Информатика данные + обработка

Носитель информации

  • Материальные объекты (книги, диски)
  • Волны (звук, свет, радио)
  • Физические состояния (атомы, молекулы)
⚠️ Нет носителя — нет информации

По способу восприятия

  • 👁️ Визуальная — зрение
  • 👂 Аудиальная — слух
  • 👃 Обонятельная — запах
  • 👅 Вкусовая — вкус
  • ✋ Тактильная — осязание
